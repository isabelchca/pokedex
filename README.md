# App Poke

Es una aplicación para consultar datos de Pokemones desde un perfil
ingresando con una cuenta de Gmail. Permite modificar nombre e imagen del perfil.
La aplicación está construida con React, JavaScript y bootstrap utilizando Firebase.

## Instalación

En el directorio del proyecto, ejecutar:

### `npm install`

Luego el comando:

### `npm start`

Abra http://localhost:3000 en el navegador

Haga click en el botón Acceder e ingrese su cuenta de Gmail
